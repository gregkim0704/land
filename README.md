## 🎯 프로젝트 소개

토지 전문 부동산 중개인을 위한 AI 기반 종합 솔루션입니다. 
복잡한 토지 분석과 투자 판단을 AI 기술로 효율화하고, 
고객에게 더 나은 서비스를 제공할 수 있도록 돕습니다.

### 타겟 사용자
- 토지 전문 부동산 중개인
- 토지 투자자
- 부동산 컨설턴트
- 토지 개발업자

## ✨ 주요 기능

### 1. 🔍 토지 종합 분석
- **자동 규제 조회**: 용도지역, 건폐율, 용적률 등 건축 규제 자동 확인
- **개발가능성 평가**: AI 기반 100점 만점 개발 가능성 점수 산출
- **시장 가격 예측**: 공시지가 기반 실거래 예상 가격 산정
- **투자 수익률 분석**: 보유 기간별 예상 수익률 시뮬레이션
- **리스크 체크리스트**: 맹지, 농지전용, 산지전용 등 주요 리스크 식별

### 2. 💬 24시간 AI 컨설팅 챗봇
- **전문가 수준 상담**: 20년 경력 컨설턴트 페르소나 기반
- **법규 및 절차 안내**: 농지법, 산지관리법, 국토계획법 등
- **세금 관련 조언**: 취득세, 양도소득세, 재산세 기본 상담
- **맞춤형 응답**: 고객의 투자 목적과 상황에 맞춘 조언

### 3. 🎯 스마트 매칭 시스템
- **고객 프로파일링**: 투자 성향, 예산, 목적 분석
- **최적 토지 추천**: AI 매칭 알고리즘으로 적합도 점수 산출
- **포트폴리오 관리**: 투자자별 토지 포트폴리오 추적

### 4. 📄 스마트 계약서 분석
- **주요 조항 자동 추출**: 계약 핵심 내용 요약
- **리스크 사전 경고**: 불리한 조항 및 법적 이슈 식별
- **체크리스트 제공**: 추가 확인 필요 사항 안내

### 5. 📊 시장 동향 리포트
- **실시간 거래 동향**: 지역별 거래량 및 가격 추세
- **투자 인사이트**: AI 기반 유망 지역 및 시기 분석
- **트렌드 분석**: 정부 정책, 개발 계획 반영

## 🏗️ 시스템 구조

```

│   ├── 공공 API 연동
│   │   ├── 국토교통부 실거래가
│   │   ├── 토지이음 API
│   │   └── 부동산공시가격
│   │
│   └── 데이터베이스
│       ├── 거래 데이터
│       ├── 고객 정보
│       └── 분석 이력
│
└── AI/ML 모델
    ├── 가격 예측 모델 (XGBoost/LightGBM)
    ├── LLM 통합 (Claude API)
    └── 시계열 예측 (Prophet)
```

## 🚀 설치 방법

### 필수 요구사항
- Python 3.9 이상
- pip 패키지 관리자
- (선택) Claude API 키 (AI 챗봇 기능 사용 시)

### 1. 저장소 클론
```bash
git clone https://github.com/your-repo/land-ai-system.git
cd land-ai-system
```

### 2. 가상환경 생성 (권장)
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate
```

### 3. 의존성 설치
```bash
pip install -r requirements.txt
```

### 4. 환경 변수 설정 (선택)
`.env` 파일 생성:
```
ANTHROPIC_API_KEY=your_api_key_here
```

## 💻 사용 방법

### 웹 애플리케이션 실행
```bash
streamlit run app.py
```

브라우저에서 자동으로 `http://localhost:8501`이 열립니다.

### 코어 모듈 직접 사용

```python
from land_ai_core import LandInfo, LandAnalyzer

# 1. 토지 정보 생성
land = LandInfo(
    address="경기도 성남시 분당구 정자동 123-45",
    land_category="대지",
    area=500.0,  # 500㎡
    official_price=3000000,  # ㎡당 300만원
    zone_type="제2종일반주거지역",
    district="일반",
    road_contact=True,
    nearest_station_km=0.8,
)

# 2. 분석 수행
analyzer = LandAnalyzer(land)
report = analyzer.generate_comprehensive_report()

# 3. 결과 출력
print(report)
```

### AI 챗봇 사용

```python
from land_ai_chatbot import LandConsultingBot

# 챗봇 초기화
bot = LandConsultingBot()

# 대화
response = bot.chat("농지를 사서 집을 지으려면 어떻게 해야 하나요?")
print(response)
```

## 📊 데모 시나리오

### 시나리오 1: 토지 분석
1. **토지 분석** 메뉴 선택
2. 토지 정보 입력 (주소, 면적, 용도지역 등)
3. **분석 시작** 버튼 클릭
4. 5개 탭에서 결과 확인:
   - 기본정보
   - 건축규제
   - 개발가능성
   - 가격분석
   - 리스크
5. 리포트 다운로드 (JSON 또는 텍스트)

### 시나리오 2: AI 상담
1. **AI 상담** 메뉴 선택
2. 질문 입력 (예: "임야 투자는 어떤가요?")
3. AI 컨설턴트의 전문적인 답변 확인
4. 추가 질문으로 심화 상담

### 시나리오 3: 고객 매칭
1. **고객 매칭** 메뉴 선택
2. **고객 등록** 탭에서 프로필 입력
   - 예산 범위
   - 투자 목적
   - 위험 성향
   - 선호 지역/지목
3. **토지 추천** 탭에서 매칭 결과 확인
4. 매칭 점수와 추천 이유 검토
