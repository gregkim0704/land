## 🎓 사용법 튜토리얼

### 시나리오 A: 처음 토지를 분석하는 경우

#### 1. 토지 정보 입력
```
메뉴: 🔍 토지 분석

입력 예시:
- 주소: 경기도 성남시 분당구 정자동 123-45
- 지목: 대지
- 면적: 500㎡ (약 151평)
- 공시지가: 3,000,000원/㎡
- 용도지역: 제2종일반주거지역
- 도로 접함: ✅
- 역까지 거리: 0.8km
```

#### 2. 분석 결과 확인
- **기본정보 탭**: 면적, 공시지가 확인
- **건축규제 탭**: 건폐율 60%, 용적률 250%
  → 약 90평 건물 신축 가능!
- **개발가능성 탭**: S등급 (최우수)
- **가격분석 탭**: 예상 시세 28.5억원
  → 투자 수익률 시뮬레이션 가능
- **리스크 탭**: 주요 리스크 체크

#### 3. 리포트 다운로드
- JSON 또는 텍스트 형식으로 저장
- 고객에게 제공 가능

### 시나리오 B: AI 컨설팅 받기

#### 1. AI 상담 메뉴 진입
```
메뉴: 💬 AI 상담
```

#### 2. 질문 예시

**초보 투자자 질문:**
```
"토지 투자 처음인데 어디서부터 시작해야 하나요?"
"농지와 대지의 차이가 뭔가요?"
"토지 살 때 무엇을 확인해야 하나요?"
```

**전문적인 질문:**
```
"농지전용부담금은 얼마나 나오나요?"
"맹지를 샀는데 진입로 확보 방법이 궁금합니다"
"양도소득세 절세 방법 알려주세요"
```

**법규 관련 질문:**
```
"개발행위허가는 어떻게 받나요?"
"용도지역별 건폐율/용적률이 궁금해요"
"산지전용 절차가 어떻게 되나요?"
```

### 시나리오 C: 고객에게 토지 추천

#### 1. 고객 프로필 등록
```
메뉴: 🎯 고객 매칭 > 👤 고객 등록

입력 예시:
- 고객명: 김투자
- 예산: 5억 ~ 20억원
- 투자 목적: 중장기 보유
- 위험 성향: 보통
- 선호 지역: 주거지역, 상업지역
- 선호 지목: 대지
```

#### 2. 토지 추천 받기
```
메뉴: 🎯 고객 매칭 > 🎯 토지 추천

결과:
⭐⭐⭐ 강력추천 (85점)
- 예산 범위 적합
- 개발가능성 우수
- 안정적 보유에 적합
- 리스크 요인 1건
```

## 💡 실무 활용 팁

### Tip 1: 복수 토지 비교
```
1. 여러 토지를 차례로 분석
2. AI 상담 메뉴에서 "분석한 토지들 중 어느 것이 더 좋나요?" 질문
3. AI가 종합적으로 비교 분석해줌
```

### Tip 2: 고객 상담 준비
```
1. 상담 전날 토지 미리 분석
2. 리포트 다운로드하여 출력
3. AI 상담으로 예상 질문 리허설
4. 고객 프로필 등록하여 매칭 점수 확인
```

### Tip 3: 계약서 검토
```
1. 계약 전 계약서 텍스트 복사
2. 📄 계약서 분석 메뉴에서 붙여넣기
3. 주요 조항과 리스크 확인
4. 고객에게 주의사항 설명
```

## 🔥 고급 활용법

### 1. 투자 수익률 시뮬레이션
토지 분석 > 가격분석 탭에서:
- 매입가 슬라이더 조정
- 보유 기간 설정
- 예상 수익률 즉시 확인

### 2. 개발 시나리오 분석
건축규제 탭 정보를 활용:
- 최대 건축 가능 규모 확인
- 용도별 수익성 계산
- 예) 근생 vs 다세대 vs 오피스텔

### 3. 리스크 기반 협상
리스크 탭 정보 활용:
- 맹지 → 가격 30% 이상 할인 요구
- 농지전용 필요 → 부담금 고려 협상
- 경사지 → 개발비용 추가 반영

## ⚠️ 주의사항

### DO ✅
- ✅ 토지 정보 정확히 입력 (측량 결과 기반)
- ✅ 현장 방문 필수 (지적도와 실제 다를 수 있음)
- ✅ AI 분석 + 전문가 검증 병행
- ✅ 법률/세무 전문가 최종 자문

### DON'T ❌
- ❌ AI 분석만으로 투자 결정
- ❌ 현장 미방문 상태에서 계약
- ❌ 공시지가만 보고 가격 판단
- ❌ 리스크 무시하고 진행

## 🆘 문제 해결 (FAQ)

### Q1: 앱이 실행되지 않아요
```bash
# 해결 방법
pip install --upgrade streamlit
streamlit run app.py
```

### Q2: 분석 결과가 이상해요
→ 입력 정보를 다시 확인하세요
→ 특히 면적 단위(㎡)와 공시지가(원/㎡) 확인

### Q3: AI 챗봇이 답변을 안 해요
→ 현재 데모 버전은 규칙 기반입니다
→ Claude API 키를 설정하면 더 똑똑해집니다

### Q4: 가격 예측이 실제와 달라요
→ 공시지가 기반 추정이므로 편차 있음
→ 인근 실거래가 직접 조회 권장

### 토지 투자 기초 지식
1. **지목의 종류와 특징**
   - 대지: 건물 부지
   - 전/답: 농경지
   - 임야: 산림

2. **용도지역 이해**
   - 주거지역 (1/2/3종)
   - 상업지역
   - 공업지역
   - 녹지지역
   - 각 지역별 건폐율/용적률 차이

   - 등기부등본
